Estructura Párrafos Enlaces Listas Multimedia Divs Formularios Metadatos Otras etiquetas

Ejemplo 1: Estructura Básica HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>UNET - Página Principal</title>
</head>
<body>
    <h1>Bienvenido a la UNET</h1>
    <p>Ejemplo de estructura básica.</p>
</body>
</html>

Explicación: Define la raíz del documento y su organización básica.

Ejemplo 2: Párrafos y Encabezados

<h1>Tipos de Frutas</h1>
<h2>Manzanas</h2>
<p>Las manzanas son rojas o verdes.</p>

<h2>Peras</h2>
<p>Las peras tienen forma de lágrima.</p>

Explicación: Jerarquía de texto con <h1> a <h6> y párrafos con <p>.

Ejemplo 3: Enlaces

<p>Visita nuestro <a href="https://unet.edu.ve">portal UNET</a>.</p>
<p>Contacto: <a href="mailto:info@unet.edu.ve">correo electrónico</a>.</p>

Explicación: Enlaces externos/internos con <a href="">.

Ejemplo 4: Listas

<h3>Cosas para pasear a mi perro:</h3>
<ul>
    <li>Correa</li>
    <li>Agua</li>
</ul>

<h3>Pasos para lavar un carro:</h3>
<ol>
    <li>Enjabonar</li>
    <li>Enjuagar</li>
</ol>

Explicación: Listas no ordenadas (<ul>) y ordenadas (<ol>).

Ejemplo 5: Multimedia

<img src="perro.jpg" alt="Perro jugando" width="200">

<video width="300" controls>
    <source src="carro.mp4" type="video/mp4">
</video>

Explicación: Inserta imágenes (<img>) y videos (<video>).

Ejemplo 6: Divs

<div style="background: #e9d8a6; padding: 10px;">
    <h3>Manzanas</h3>
    <p>Texto sobre manzanas.</p>
</div>

Explicación: Contenedores para agrupar elementos.

Ejemplo 7: Formularios

<form action="/procesar" method="POST">
    <label for="fruta">Fruta favorita:</label>
    <input type="text" id="fruta" name="fruta">

    <input type="submit" value="Enviar">
</form>

Explicación: Campos interactivos para usuarios.

Ejemplo 8: Metadatos

<head>
    <meta charset="UTF-8">
    <meta name="description" content="Página UNET">
    <title>Ejemplo</title>
</head>

Explicación: Información invisible para navegadores.

Ejemplo 9: Otras Etiquetas

<p>El <strong>perro</strong> es <em>muy</em> juguetón.</p>
<details>
    <summary>Ver más</summary>
    <p>Las peras son dulces.</p>
</details>

Explicación: Etiquetas semánticas y de formato.
